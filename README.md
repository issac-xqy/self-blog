## ⚡ 快速开始

<p align="center">
  <img src="https://raw.githubusercontent.com/Tarikul-Islam-Anik/Animated-Fluent-Emojis/master/Emojis/Hand%20gestures/Rocket.png" width="40" />
</p>

<details open>
<summary><b>⚙️ 启动 & 使用</b></summary>

```bash
# ① 克隆仓库
git clone <your-repo-url> && cd <your-repo>

# ② 安装依赖
npm install

# ③ 启动开发服务器  →  http://localhost:4321
npm run dev

# ④ 写一篇新文章
npm run new "我的第一篇文章"

# ⑤ 构建 & 预览
npm run build
npm run preview
```

</details>

<details>
<summary><b>📦 可用脚本一览</b></summary>

| 命令 | 说明 |
|------|------|
| `npm run dev` | 启动 Astro 开发服务器，支持 HMR 热更新 |
| `npm run build` | 编译生产版本 + 运行 Pagefind 搜索索引 |
| `npm run preview` | 本地预览构建后的 production 产物 |
| `npm run new "标题"` | 在 `src/content/blog/` 下创建带 frontmatter 的 MDX 模板 |

</details>

## 📝 文章格式

编辑 `src/content/blog/你的文章.mdx`：

```yaml
---
title: "文章标题"
description: "文章摘要 — 会出现在卡片和 SEO 描述中"
pubDate: 2026-05-20
updatedDate: 2026-05-21  # 可选
tags: ["标签1", "标签2"]
coverImage:                # 可选：文章封面图
  src: "/images/cover.png"
  alt: "封面描述"
draft: false               # true = 草稿，构建时跳过
---

从这里开始写 Markdown 内容…

## 支持 Markdown 语法

- 代码块自动高亮
- 表格、引用、列表
- **粗体**、*斜体*、~~删除线~~
- [链接](https://example.com)
```

> **💡 提示**：执行 `npm run new "标题"` 会自动生成带好 frontmatter 的空白文章模板。
